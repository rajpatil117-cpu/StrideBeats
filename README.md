# 🚶 StrideBeats

StrideBeats is a modern fitness companion that combines walking, adaptive music recommendations, and dynamic weather visuals into one immersive experience.

## Vision

To make walking enjoyable by creating a fitness app that reacts to the world around you.

## Current Features

- Walking dashboard
- Daily goal tracking
- Step counter (Demo)
- Calories
- Distance
- Active time
- Weather-aware UI
- Music recommendations
- Glassmorphism interface

## Upcoming Features

- Real step tracking
- Live weather API
- GPS distance
- Walking history
- Achievements
- AI-powered recommendations

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
